# How To Use
```lua
-- loading GRH-Service
loadstring(game:HttpGet("https://raw.githubusercontent.com/GrblxHOfficial/GRH-Service/main/source.lua"))()

-- GRH-Service Syntax
GRH.Fly.Enabled = bool
GRH.Fly.Speed = number

GRH.Noclip = bool

GRH.InfJump = bool

GRH.FindNearestPlayer() -- function that returns nearest player to your character

GRH.ws = number -- change walkspeed

GRH.jp = number -- change jump power

GRH.lp -- local player

GRH.c -- local player character

GRH.m -- local player mouse

GRH.Gravity = number

GRH.Aimbot.Enabled = bool
GRH.Aimbot.AimAtPlayer = Player

GRH.FindNearsetPlayerToMouse() -- functon that returns nearest player to mouse

```
